## 🛠️ Instalação e uso

1. **Clone o repositório**:
```bash
git clone https://github.com/suzianefontenele/portal-t2-stark.git
```

2. **Instale as dependências:**:
```bash
npm install
```

3. **Execute o projeto em modo desenvolvimento:**:
```bash
npm run dev
```

4. **Build para produção:**:
```bash
npm run build
```

5. **Preview do build:**:
```bash
npm run preview
```
